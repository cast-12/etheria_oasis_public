Firebase setup required

Connect a free Firestore database so all guild members share the same counter data in real time.

  1. Go to console.firebase.google.com → Create a project (free).
  2. Inside the project click Firestore Database → Create database → Start in test mode.
  3. Go to Project Settings (gear icon) → Your apps → click </> web icon → Register app → copy the config values below.

Config is saved in your browser. For Vercel deployment, hardcode your values in the HARDCODED_CONFIG variable in the script so all visitors connect automatically.
